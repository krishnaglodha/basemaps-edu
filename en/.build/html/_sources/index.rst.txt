GeoSolutions BaseMaps  
==========================
**GeoSolutions BaseMaps** is an **Open Source** solution built on top of Docker to create own basemap using OSM data, Geoserver, PostgreSQL 

This training aims to introduce the attendees to all the aspects of **GeoSolutions BaseMaps** whether they are novice or expert users.
