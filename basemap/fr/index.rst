GeoServer Modules d'Apprentissage Avancé
========================================
**GeoServer** est un **Open Source**, un Logiciel Libre, de géstion écrit en langage Java qui permit à l'utilizateur de partager et éditer les datas provénant des systèmes d'informations géographiques (geospatial). Conçu pour l'interoperabilité, il publie les datas provenant des majeures fonts spatial utilizant les standards ouverts.

Ce module d'apprentissage a l'objectif d'introduiser l'utilizateur à tous les aspects q'on pourrait rencontrer pendant l'utilization de **GeoServer 2.2.x** quoi q'on soit des experts ou des novices de ce programme.
