Travailler avec OpenLayers
==========================

**OpenLayers** facilite à mettre une carte dynamique dans une page web. La bibliothèque permet les développeurs d'integrer des données provenant de différent sources, offre un accueillant API, et ensuit en engageant et sensibles applications de cartographie. 

Cette module introduit les concepts fondamentaux de OpenLayers pour créer et gérer une carte.
