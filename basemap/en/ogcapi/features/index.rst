OGC API - Features
==================

This module introduces the OGCAPI - Features service, in particular shows how to obtain vector data information, filter, extract and reproject it.
It provides also informations about the direction taken by OGC in defining the standard for Features Transaction, that are currently not supported by OGC API - Features implementation of GeoServer.
